# Frontend (React)

This is the React dashboard for the Business Filing Tracker. It connects to the Rails API backend and provides a clean, readable UI for tracking business filings.

## Features
- Dashboard layout with summary cards
- Table of filings with status pills
- Add new filing form
- Status filter chips
- Update status dropdowns
- Human touch: paperwork goblin empty state

## Setup
1. `npm install`
2. `npm run dev`
3. Visit [http://localhost:5173](http://localhost:5173)

## Notes
- API base URL defaults to `http://localhost:3000`
- No auth, no cloud, no paid services
- Styling is intentionally simple and professional
